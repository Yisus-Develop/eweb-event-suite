Advanced Event Management System for WordPress, designed with a decoupled architecture using **Repositories**, **Providers**, and automated **Cron Dispatchers**. Part of the **EWEB Plugin Suite**.

## 🏗️ Architecture & Level

This plugin is built following modern PHP and WordPress patterns:
- **Repository Pattern:** Centralized data access for events, subscribers, and queues.
- **Provider Pattern:** Decoupled email delivery system (supports LocalLog and external providers).
- **Dual Persistence:** High-reliability data storage.
- **Automated Dispatcher:** Built-in Cron system for event notifications.

## 🚀 Key Features

- **Country-Based Routing:** Manage events and subscriptions by geographical location.
- **Subscriber Management:** Integrated with Contact Form 7 for automatic lead capture.
- **Dynamic Shortcodes:** Custom buttons, event lists, and popup triggers.
- **Change Detection:** Automatically detects event updates and schedules notifications.
- **Admin Dashboard:** Full management UI for event settings and logs.

## 🛠️ Installation

1. Upload the `mc-event-suite` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure settings in the **MC Event Suite** menu.
